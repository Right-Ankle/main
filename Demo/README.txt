
　2020年度卒論における提案手法のデモ

　実行環境：Windows10

　実行手順：

　　・Demo.exeをクリック

　　・使用画像を選択
　　　(ex. 1を入力してEnter -> Barbara.bmp)

　　・使用したい領域分割法を選択
　　　step1を用いた領域分割を行う場合 -> y
 　　 先行手法の領域分割を行う場合 -> n

　出力：

　　・コマンドプロンプトに，DCTの符号化レートに対応した領域分割後のエントロピーとPSNR

　　・OUTPUT内に領域分割結果の画像（符号化レート.bmp）

　　・ICA単独での再構成画像(ICA_SAI.bmp)

　　・ICA基底の画像（ICA_BASIS.bmp）
